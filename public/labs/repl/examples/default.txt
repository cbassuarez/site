// this is seb's repl, a cybernetic art engine
// wildcards choose samples; attractors act as cybernetic filters
//
// notation:
//   |       bar divider
//   .       rest
//   ~       sustain previous note / hold previous value
//   (...)   subdivide one slot's time evenly
//   ;       gate sound to end of rhythmic unit (otherwise sounds are one-shots irrespective of rhythmic value end)
//
// wildcards:
//   *       random pitch / sample / parameter value
//   *!      frozen random value
//   *4      random pitch in octave 4
//   *!4     frozen random pitch in octave 4
//   A*      random octave of A
//   C*!     frozen random octave of C
//
// control streams:
//   *       random legal value
//   ~       hold previous resolved value
//   _       reset to default
//   *&16    drift between random values over 16 seconds
//
// samples:
//   BANKNAME-*        random sample from BANKNAME bank
//   BANKNAME-*&24     crossfade through BANKNAME bank over 24 seconds
//   BANKNAME-*&24;    same, but gated to the current slot/subdivision (does not continue playing after rhythmic unit is done)
//
// attractors:
//   attractor weather.dew  colors literal material as humid pressure
//   attractor archive      biases sample memory / aging / lineage
